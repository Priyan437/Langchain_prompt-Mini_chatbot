### Features

- It has a Mini Chat bot which is general in case.
- It has Different Prompting techniques implemented chat Models


Clone the project

```bash
https://github.com/Priyan437/Langchain_prompt-Mini_chatbot.git
```

Go to the project directory

```bash
  cd Langchain_prompt-Mini_chatbot
```

Create and Activate the Virtual Enviroment

```bash
python -m venv prompt
# Windows:
.\prompt\Scripts\activate
# Linux/macOS:
source prompt/bin/activate
```


Install dependencies

```bash
  pip install requirements.txt
```
